# 教案后端说明
## 持久层部署（MySQL）
暂时无表需求。存在database即可
## 执行逻辑概述
- HTTP请求 -> [ChatController](src/main/java/com/chongcc/test/Controller/AI/ChatController.java) 
-> [AIChatService](src/main/java/com/chongcc/test/Service/AI/AIChatService.java)
-> [火山引擎大模型接口](https://www.volcengine.com/docs/82379/1399009?lang=zh) 
## [配置文件yml](src/main/resources/application.yml)
    spring:
      application:
        name: test
      datasource:
        url: jdbc:mysql://localhost:port/database_name?useSSL=false&allowPublicKeyRetrieval=true&
        username: username
        password: your_password
      jpa:
        hibernate:
          ddl-auto: validate
          naming:
            physical-strategy: org.hibernate.boot.model.naming.PhysicalNamingStrategyStandardImpl
        show-sql: true
    # 火山引擎
    volc:
      api-key: 填写火山引擎申请的api-key
      img-model: doubao-seedream-3-0-t2i-250415
      chat-model: doubao-seed-1-6-250615
      prompt: ",以米字格字帖样式生成"
该部分为教案所需配置部分，创建.yml文件并粘贴内容填写必须参数
## 使用.jar文件运行后端
- `java -jar .\target\test-0.0.1-SNAPSHOT.jar --spring.config.location=./application.yml`
`.\target\test-0.0.1-SNAPSHOT.jar`与`./application.yml`根据情况自己更换路径
